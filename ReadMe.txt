

Installation
   npm v4.18.2
   mongoose v6.5.2
   bootstrap": "^5.2.0",
   express": "^4.18.2",

Download and install Node.js  express
$npm install
$npm install express

Install mongoDB  via command line
$ brew install mongodb-community@ -version
Run application:
 use the command $ npm start to start application
 ctrl+c to stop running application
